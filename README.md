A simple Todo List application built with **React**, **TypeScript**, and **Vite**.  
This project demonstrates the use of **components**, **hooks**, and **CSS styling** to create a functional task manager.

## Features

- Add new tasks
- Edit tasks inline
- Delete tasks
- Responsive and clean UI with CSS

## Technologies Used

- React 18
- TypeScript
- Vite
- CSS

## How to Run Locally

1. **Clone the repository:**

```bash
git clone https://github.com/amanabate/A2sv-webTrack-task-4.git

cd <A2sv-webTrack-task-4>

npm install

npm run dev


## Project Structure
src/
  main.tsx          # App entry point
  App.tsx           # Main component
  types.ts          # Type definitions
  components/
    TodoInput.tsx   # Component for adding tasks
    TodoList.tsx    # Component for displaying list of todos
    TodoItem.tsx    # Component for each todo item
  index.css         # Global CSS
  styles.css        # App-specific styling

```
