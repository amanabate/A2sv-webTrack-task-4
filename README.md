A simple Todo List application built with **React**, **TypeScript**, and **Vite**.

## Features

- Add new tasks
- Edit tasks inline
- Delete tasks
- Mark tasks as completed
- Persist tasks using `localStorage`
- Clear all or remove completed tasks
- Responsive and clean UI with CSS

## Technologies Used

- React 18
- TypeScript
- Vite
- CSS
